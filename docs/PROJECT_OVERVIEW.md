# Project Overview

## What this is
A standalone proposal website for the **Travel eSIM** direct-marketing initiative. It presents strategy, scope, assumptions, and recommendations in a single shareable web experience.

## Target audience
- Internal decision-makers and stakeholders reviewing the proposal
- External partners invited to review the direction
- Intended for authenticated link-sharing only (not public SEO)

## Deployment environment
- **Hosting:** Vercel
- **Framework:** Next.js (App Router)
