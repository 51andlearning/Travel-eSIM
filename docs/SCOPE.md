# Scope

## Pages included
- `/` — Home / proposal landing
- `/strategy` — Strategic approach (placeholder)
- `/execution` — Execution plan (placeholder)
- `/appendix` — Assumptions, sources, and references (placeholder)

## Sections included
The home page is composed of modular sections under `src/components/sections/`:
- Hero
- Executive Summary
- Opportunity / Problem framing
- Strategy Pillars
- Channel Mix
- Measurement Plan
- Timeline
- Investment / Commercials
- Team & Ownership
- Assumptions & Caveats
- Call to Action / Next Steps

## Explicitly excluded
- Creative production (ad units, video, copy variants)
- Legal and privacy review deliverables
- Localization and translation
- CRM / martech implementation
- Paid media buying or trafficking
- Live analytics dashboards or BI tooling
- Customer-facing forms with data capture (contact form is static only for v1)

## Launch milestones
1. **M1 — Scaffold complete:** Project created, shadcn primitives installed, docs drafted. *(current)*
2. **M2 — Content draft:** All sections populated with placeholder content from the approved deck.
3. **M3 — Stakeholder review:** Internal review pass, feedback cycle.
4. **M4 — Deploy preview:** Published to Vercel preview URL for stakeholder sharing.
5. **M5 — Approved build:** Final content, tracking enabled, production URL live.
6. **M6 — Post-launch:** Analytics review, iteration cycle.
